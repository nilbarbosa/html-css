com medo dos avanços tecnológicos da União Soviética, em 1969 os Estados Unidos imforma a criação de uma agência que desenvolveria um projeto para proteger os dados e documentos importantes do país, mal sabiam eles que criariam uma grande revolução da vida na terra! Estamos falando da internet.



A escrita não semântica era ultilizada no HTML4 e possibilitava adicionar tegs de estilo dentro do html. com a evolução da linguagem, os desenvolvedores foram tirando essa opção com o intuito de separar melhor o HTML do CSS deixando as opções de estilo exclusivamente para o CSS priorizando o uso das tegs em html exclusivamente para dar sentido. Como por exemplo, se você quer da enfaze a algum texto, você simplesmente coloca a teg <em>, que dirá ao desenvolvedor css que aquela frase precisará receber emfase, e cabe ao profissional desenvolver uma forma de comprir isso usando tegs de estilo.
